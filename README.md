# easeofaccess
Control Panel for Accessability Features


Ease of Access aims to bridge the gap between traditional accessibility tools and modern AI‑driven interaction, giving users new ways to navigate and control their system when standard I/O devices fail or users that find it diifucult to use them properly in the first place.
The application abstracts and unifies accessibility features into a clean WinUI3/WPF modular system that functions as an integrated background process to adapt to different accessability requirements without overwhelming the user. 

---
Key Features 
- **Dynamic hotkey and system‑level input hooks**, powered by Win32 APIs for fast, customizable control
**AI‑powered interaction**, enabling users to operate their computer through physical movement or alternative inputs — aimed for bypassing broken I/O devices or supporting users with disabilities
- **Integrated tiling window manager**, giving users a keyboard‑driven layout system that keeps workflows fast, organized, and accessible
- **On‑demand tool windows**, instantly summonable panels that streamline tasks, reduce friction, and keep essential utilities always within reach
- **I/O recovery tools**, such as keyboard‑based mouse control and key remapping, allowing users to regain functionality when hardware fails
- **Secure profile syncing**, enabling quick setup on any machine by loading user preferences through authenticated access
